# Syntax

```bash
shutdown [OPTIONS...] [TIME] [WALL...]
# options
-r # reboot 
-h # shutdonw and power-off
-c # cancel shutdown 
# time 
+5: # 5分钟后执行
22：00 # 到指定时间执行
```
