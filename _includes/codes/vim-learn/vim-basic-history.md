# Vim 键知识

## 电脑通用
- 回车键: 现在用 Enter, 以前用过 CR(Carriage Return), Return(Mac OS系统叫法). 来源：来源于旧式机械英文打字机，打字机有一个部件称为“字车”， 每打一个字符，字车就前进一格，打满一行字符后，打字者需要推动字车到起始位置，这时打字机会有两个动作: 字车归位+滚筒上卷一行, 这个推动字车的动作叫做“回车”.
