# 电

## 弱电

### 网络

- Mesh路由器+有线网络回程


#### 网络布线知识
- 超五类线在线路不长(如100平米房子中的布线), 可以达到万兆

>网速上不去：检查是否是接触不良，因为跑1Gbps需要用网线里8根线，100Mbps只需要用1、2、3、6四根。如果4、5、7、8有接触不良，自动降为百兆
